# Tic Tac Toe

> The very first app I ever wrote

### Installation:

```sh
$ npm install http-server -g
$ npm install
$ cd tic-tac-toe
$ grunt
$ http-server
```

> [http://localhost:8080/](http://localhost:8080/)
